수학적 하트 방정식을 활용해 직접 제작하였다.  
WordCloud 라이브러리를 기본으로 하고, 아래의 **하트 형태 마스크 생성 부분**이 핵심이다.

### 🧠 하트 마스크 생성 코드 (핵심 부분만)

```python
# 4) 하트 모양 마스크 만들기
W, H = 1200, 900
y, x = np.ogrid[-1.4:1.4:H*1j, -1.4:1.4:W*1j]

eq = (x**2 + y**2 - 1)**3 - x**2 * (y**3)

# 🔥 '하트 내부'만 글자가 채워지도록 영역 설정
heart_inside = eq > 0

# WordCloud 규칙: 0이 아닌 값(=255)이 글자를 찍을 수 있는 공간
mask = np.where(heart_inside, 255, 0).astype(np.uint8)
```
